Project Overview

This project was a platformer that was built in Unreal Engine 5, for the purpose of showcasing Level Design. It was made in a group of three.

In the first video, my role was the gameplay designer. I created the doors that required a code to get through, and I did this using the widget and interface blueprints in UE5. Then, I created platforms that occasionally disappeared for the player to traverse

In the second video, my role was environment artist. I mainly focused on lighting and post-processing in order to try and create natural light, but since the level was indoors I had to focus on indirect lighting such as lamps and wall posts. I also used Lumen to help and heighten the lighting.
